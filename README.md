# Beyond Group - Foam Quote Tool

A professional calculator for bridge abutment rehabilitation projects. Developed as part of the AI Automation Developer Challenge.

## Quick Start
1. Ensure you have Node.js installed.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open the tool in your browser (defaults to http://localhost:5173).

## Key Features
- **Accurate Billing**: Revenue calculated on Raw Lbs (before waste) as per company policy.
- **Internal Visibility**: Real-time margin and cost build analysis using Drum Cost figures ($2,300/set).
- **Project Specifics**: Built-in support for Mobilization fees, Extended Day rates ($750/day), and Travel time labour.
- **Inflation Protection**: Immediate calculation of 2027 contingency pricing using 4% compounded escalation.

## Scenario Breakdown: Lethbridge Job
This tool successfully calculates the following for the 18 m³ Lethbridge void:
- **Material Yield**: 2354.4 lbs
- **Total Revenue**: $32,807.20
- **2027 Contingency**: $34,119.49
